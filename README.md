
# 🌦️ Securion Weather API

Spring Boot REST API for importing, storing, filtering, sorting, and retrieving weather data from CSV files.

---

## 🚀 Tech Stack

- Java 17+
- Spring Boot 4
- Spring Data JPA
- MySQL
- Maven
- REST APIs
- CSV Data Import
---
## 📦 Features
✅ CSV Weather Data Import  
✅ REST API Architecture  
✅ Sorting Support  
✅ Filtering Support  
✅ Pagination (Large Dataset Handling)  
✅ DTO Layer  
✅ Global Exception Handling  
✅ MySQL Persistence  
---
## 🗄️ Database

**Database Name:** `weather_db`  
**Table:** `weather_data`
Columns:
- id
- weather_date
- temperature
- humidity
- pressure
- heat_index
- weather_condition
